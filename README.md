AR Dashboard (Web AR Project)
About this project-

This is a small project I built to understand how augmented reality works on the web.

The idea is simple — when the camera detects a specific image, a 3D model appears on top of it. I made this mainly for learning and demo purposes, especially for academic use.

It runs directly in the browser, so no need to install any app.

- - - - -

What it does- 
Detects image targets using the camera
Displays 3D models on top of those images
Works in the browser
Has a simple dashboard UI to access the AR experience

- - - - -

Tech used
HTML
CSS
JavaScript
Web-based AR (image tracking)

- - - - -

How to run

You can run this in two ways:

1. Using Live Server (recommended)

Open the project in VS Code
Run it with Live Server

2. Directly

Open index.html in your browser

Note: Camera access might not work properly without a local server.
